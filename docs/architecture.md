# Technical Architecture

## The Stack
- **Backend**: FastAPI (Python) - High performance, automatic docs.
- **Frontend**: React + Vite (JS) - Fast development, component-based.
- **Styling**: Tailwind CSS - Clean, responsive design.
- **Database**: SQLite (Dev) / PostgreSQL (Prod).
