# Development Roadmap (8-Week Polished MVP)

## Phase 1: Foundation (Weeks 1-2)
- **Week 1: Infrastructure**: Set up FastAPI structure, PostgreSQL connection, and React + Vite environment.
- **Week 2: Auth & Security**: Implement JWT tokens, password hashing, and Login/Signup pages.
- **Deliverable**: A secure system where users can log in to a blank dashboard.

## Phase 2: Core CRM (Weeks 3-4)
- **Week 3: Lead Management**: Build the Lead CRUD (Create, Read, Update, Delete) and the filterable Lead List table.
- **Week 4: Interaction Timeline**: Build the Lead Detail page and the activity logging system (Calls/Emails).
- **Deliverable**: A working CRM where you can add leads and track conversations.

## Phase 3: Visualization & Tasks (Weeks 5-6)
- **Week 5: Kanban & Dashboard**: Build the drag-and-drop Kanban board and the stats cards.
- **Week 6: Task System**: Build the to-do list and the "Convert Lead to Client" workflow.
- **Deliverable**: A visual pipeline that feels like a professional SaaS product.

## Phase 4: Polish & Deploy (Weeks 7-8)
- **Week 7: UI/UX Pass**: Add loading spinners, "Toast" notifications, and mobile responsiveness.
- **Week 8: Documentation & Launch**: Final testing, API documentation, and deployment to Render/Vercel.
- **Deliverable**: A production-ready CRM live on the web.
