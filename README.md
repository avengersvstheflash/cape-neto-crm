# Cape Neto CRM

A lightweight Customer Relationship Management (CRM) system built for Cape Neto.
Developed as an internship project using **FastAPI (Python)** for the backend and **React (Vite)** for the frontend.

---

## Project Status

> Currently in **initialization phase** — foundation and documentation only. No business logic implemented yet.

---

## Tech Stack

| Layer      | Technology          |
|------------|---------------------|
| Backend    | FastAPI (Python)    |
| Database   | SQLite (MVP phase)  |
| Frontend   | React + Vite        |
| Styling    | Tailwind CSS        |

---
